# Running Your Site Locally

1. Install [Node.js and npm](https://nodejs.org/en/)

2. Install npm dependencies:

        npm install

3. Start the Gatsby local development server:

        npm run develop

4. Browse to [http://localhost:8000/](http://localhost:8000/)
